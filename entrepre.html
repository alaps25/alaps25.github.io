In 2012, Bangalore was already experiencing traffic problems and we wanted to build a solution before it gets worse. We were trying to figure out P2P solutions which would not require huge initial investments. There were many carpool apps around the world but nobody was solving the end to end problem for the end user. We built a solution which helps user start to end for share rides more effectively. The solution won The 1776 Challenge Cup Delhi competition and was selected for finals in Washington DC.
